# 📉 Statistical Validation

## Business Question

**Does Discount have a statistically significant relationship with Profit?**

### Null Hypothesis (H₀)

There is no statistically significant relationship between Discount and Profit.

### Alternative Hypothesis (H₁)

There is a statistically significant relationship between Discount and Profit.

### Statistical Method

- Pearson Correlation Analysis
- Python (Pandas & SciPy)

### Results

| Metric | Value |
|---------|-------|
| Correlation Coefficient (r) | **-0.2195** |
| P-value | **2.70 × 10⁻¹⁰** |
| Significance Level | **0.05** |
| Decision | **Reject H₀** |

### Interpretation

The analysis identified a **statistically significant weak negative relationship** between **Discount** and **Profit**. This indicates that higher discounts are generally associated with lower profitability, emphasizing the importance of optimized pricing and discount strategies.
